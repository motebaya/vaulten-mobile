## Vaulten v2.1.0

Initial public release of Vaulten — a fully offline, privacy-first credential manager for Android.

### Highlights

- **AES-256-GCM encryption** for all stored credentials with a DEK/KEK key hierarchy
- **BIP-39 passphrase generation** — cryptographically secure 12-word mnemonic
- **Android Keystore integration** — hardware-backed key storage when available
- **PBKDF2-HMAC-SHA256** key derivation with 310,000 iterations
- **6-digit PIN** with bcrypt hashing (12 rounds) and biometric unlock support
- **Auto-lock** on background, screen-off, and 5-minute idle timeout
- **Screenshot prevention** via FLAG_SECURE

### Credential Types

- Standard accounts (username, password, email, notes)
- Social media (phone, 2FA, recovery options)
- Crypto wallets (private key, seed phrase)
- Google accounts (full recovery field support)

### Backup & Portability

- Encrypted `.zip` export with custom `vault.enc` binary format
- Cross-device import with backup preview and passphrase verification

### Architecture

- Kotlin 2.0 · Jetpack Compose · Material 3
- MVVM + Clean Architecture · Hilt DI · Room database
- Per-ABI split APKs: `armeabi-v7a`, `arm64-v8a`, `x86_64`, and universal
